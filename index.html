Libraries/janus

A web application that redirects users to various complex URLs according to a simple, consistent API.

Janus Demo

Pros

  • JavaScript: Commonly-known and -used.
  • node.js's npm: Similar to Ruby's Bundler and PHP's Composer.
  • Browserify: Use npm modules in node and/or the client.
  • Web scraping: Very easy with node and Cheerio.

url-js Demo

To test URLs, Libraries/url-js makes HTTP requests of them with node, and scrapes their responses with Cheerio: "Fast, flexible, and lean implementation of core jQuery designed specifically for the server." Much easier than scraping with XPath combined with some other server-side language.
